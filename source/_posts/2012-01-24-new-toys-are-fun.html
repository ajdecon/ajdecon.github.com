As of about a month ago at work, we got a shiny new cluster running Intel's new Sandy Bridge processors. We immediately started running Windows HPC on them, which made me sad in way, but what the user needs for the application is right by definition.  We got some nice improvements even with code that couldn't be re-compiled for the new optimizations available, but we finally got to start cutting a few of them over to Linux today. I can't wait to start running benchmarks...
